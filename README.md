# senderwallet-registrar
SenderWallet User Registrar Contract

## Build
In `contracts` Dir: 
- Build all artifacts: `make`


## Test
In `contracts` Dir: 
- Run all tests: `make test`
- Run with some flags:
  - Run specific test file: `TEST_FILE={filename} make test`
  - Print contract logs: `LOGS=1 make test`
